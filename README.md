<h1> What Timers are these? </h1>
<p>Esse é o meu primeiro projeto pessoal feito usando o framework <a href="https://vuejs.org/">Vue.</a></p>
<p>
  Consiste em vários temporizadores diferentes: Um para contar voltas, outro para contagens regressivas
  e um para a aplicação da técnica pomodoro.
</p>
<p>Foi estudado para esse projeto tudo que envolve o básico do framework, incluindo gerenciamento de estado
métodos de ciclo de vida, componentes, transições. Também foi usado o <a href="https://router.vuejs.org/">vue-router</a> para o roteamento da aplicação</p>
<h2>Como instalar?</h2>
<p>Clone este repositório. Na pasta clonada, execute pelo terminal o comando <code>yarn</code> ou <code>npm install</code> para instalar as depedências. Feito isso, você pode rodar a aplicação com o comando <code>yarn serve</code> ou então <code>vue-cli-service serve</code></p>
